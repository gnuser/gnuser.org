Lua是什么

  • Lua是一种编程语言,强大,快速,轻量级,可嵌入的脚本语言.
  • 语法很简单,速度却很快,很多游戏公司都在使用lua,因为魔兽世界这款游戏使用了Lua, 导致业界很多公司都开始使用lua,包括网易等国内一线游戏公司
  • 官网http://www.lua.org

        -- e1为初始值, e2为结束值, e3为每次循环增加的数值

第三个程序

  • 上面提到了变量 
    i = 1
print(i)
  • 修改变量值 
    i = 1
print(i)
i = 5
print(i)
  • 变量名字
    • 使用容易理解的英文单词,可以添加下划线来区别单词, 可以添加数字来区别
      • 比如name, myname, yourname
      • 可以使用大写字母Name, MyName, YourName, 一般习惯于不同单词的首字母是大写的
      • 可以使用下划线name_1, my_name, your_name
      • 使用数字name1, name2, name3
      • 注意,命名一定要准确,容易理解,就像写文章一样,用词和标点都是表现作者功底的一部分
    • 糟糕的命名,错误的命名举例
      • 无意义的单词: fdfa, a, b, c, a1, v1
      • 以数字开头的(直接程序报错): 1a, 2a, 3a, 1name
    • 变量作用区域 (重要) 
      i = 1
function test()
    i = 2
    print(i)
end
test()
print(i)
    • 这里用到了函数,以及如何调用函数
    • local,局部变量的作用 
      i = 1
function test()
    local i = 2
    print(i)
end
test()
print(i)
    • 没有函数的概念,没有变量的概念,接下来讲解

function keyboardEnter(key)
    if key == "A" then
        print("you entered A")
    elseif key == "B" then
        print("oh, you entered B")
    elseif key == "C" then
        print("ah, you entered C")
    else

    end
end
